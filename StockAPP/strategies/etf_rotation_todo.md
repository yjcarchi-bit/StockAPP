## 功能完成清单

### 高优先级 ✅

- [x] **ATR动态止损功能**
  - 参数: atr_period=14, atr_multiplier=2.0
  - 支持固定止损和跟踪止损两种模式
  - 防御ETF豁免ATR止损

- [x] **短期动量过滤功能**
  - 参数: short_lookback_days=12, short_momentum_threshold=0.0
  - 排除短期动量不足的ETF

- [x] **近期大跌排除逻辑**
  - 参数: loss_threshold=0.97 (3%阈值)
  - 检查近3日是否有单日跌幅超过阈值

### 中优先级 ✅

- [x] **防御ETF机制**
  - 参数: use_defensive_etf=True, defensive_etf="511880"
  - 当无合格ETF时，选择防御ETF

- [x] **得分阈值过滤**
  - 参数: max_score_threshold=6.0, min_score_threshold=0.0
  - 排除得分异常的ETF

### 低优先级 ✅

- [x] **可选过滤器**
  - MA均线过滤 (默认关闭)
  - RSI过滤 (默认关闭)
  - MACD过滤 (默认关闭)
  - 成交量过滤 (默认关闭)
  - 布林带过滤 (默认关闭)
