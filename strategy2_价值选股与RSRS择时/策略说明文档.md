# 价值选股与RSRS择时策略

## 一、策略简介

想象你是一个投资者，想要在股市中赚钱，但不知道什么时候买、买什么。这个策略就是为了解决这两个核心问题：

- **买什么？** → 选择"物美价廉"的好公司（价值选股）
- **什么时候买？** → 等市场趋势向上时再买（RSRS择时）

---

## 二、选股逻辑：寻找"物美价廉"的公司

### 2.1 什么是"物美价廉"？

在股市中，我们可以用两个指标来判断一家公司是否"物美价廉"：

| 指标 | 名称 | 含义 | 策略偏好 |
|------|------|------|----------|
| PB（市净率） | 股价 ÷ 每股净资产 | 花多少钱买公司的净资产 | 越低越好（便宜） |
| ROE（净资产收益率） | 净利润 ÷ 净资产 | 公司用股东的钱能赚多少 | 越高越好（能赚钱） |

### 2.2 打个比方

假设有两家餐厅：

**餐厅A**：净资产100万，每年赚20万，股价200万
- PB = 2倍，ROE = 20%

**餐厅B**：净资产100万，每年赚5万，股价300万
- PB = 3倍，ROE = 5%

显然，**餐厅A更划算**：价格更低（PB低），赚钱能力更强（ROE高）。

### 2.3 选股步骤

1. 从全市场筛选出PB>0且ROE>0的公司（排除亏损公司）
2. 按PB从低到高排名（便宜的排前面）
3. 按ROE从高到低排名（能赚钱的排前面）
4. 把两个排名相加，得到综合排名
5. 选择综合排名最好的**10只股票**

## 三、择时逻辑：RSRS指标

### 3.1 什么是RSRS？

RSRS的中文名叫"阻力支撑相对强度"，通过分析股票最高价和最低价的关系，判断市场的"阻力"和"支撑"谁更强。

### 3.2 通俗理解

想象一个人在爬山：

- **最高价** = 他能爬到的最高点
- **最低价** = 他滑落的最低点

如果每次滑落的最低点都在变高，说明这个人在**向上爬**（趋势向上）。如果每次爬到的最高点都在变低，说明这个人在**往下滑**（趋势向下）。

### 3.3 RSRS数值含义

| RSRS数值 | 市场状态 | 操作建议 |
|----------|----------|----------|
| 大于 0.7 | 市场强势，趋势向上 | 买入信号 |
| 小于 -0.7 | 市场弱势，趋势向下 | 卖出信号 |
| -0.7 至 0.7 | 市场震荡，方向不明 | 观望 |

### 3.4 形象比喻

把RSRS想象成一个"红绿灯"：

- **绿灯（RSRS > 0.7）**：可以通行，买入股票
- **红灯（RSRS < -0.7）**：禁止通行，卖出股票
- **黄灯（-0.7 至 0.7）**：减速慢行，保持现状

## 四、策略完整流程

| 步骤 | 条件 | 操作 |
|------|------|------|
| 1 | 计算RSRS指标 | 获取市场趋势信号 |
| 2 | RSRS > 0.7 | 选股并买入 |
| 3 | RSRS < -0.7 | 清仓卖出 |
| 4 | RSRS在-0.7至0.7之间 | 保持现状 |
| 5 | 重复步骤1-4 | 等待下一个交易日 |

## 六、策略优缺点

### 优点

| 优点 | 说明 |
|------|------|
| 选股有依据 | 基于财务指标，选择真正有价值的公司 |
| 择时有信号 | RSRS指标提供明确的买卖时机 |
| 分散风险 | 持有10只股票，避免单只股票风险 |
| 纪律性强 | 严格按照信号操作，避免情绪干扰 |

### 缺点

| 缺点 | 说明 |
|------|------|
| 震荡市表现一般 | 在横盘震荡时可能频繁买卖 |
| 信号滞后 | RSRS是趋势指标，可能错过最佳买卖点 |
| 不适合短线 | 策略偏向中长期，短线效果不佳 |

## 八、常见问题

**Q1：为什么选10只股票？**

A：分散风险。太少风险集中，太多管理困难，10只是经验值。

**Q2：RSRS的0.7和-0.7是怎么来的？**

A：这是统计学的标准分。0.7表示比平均值高0.7个标准差，属于较强的信号。

**Q3：如果RSRS一直在-0.7到0.7之间怎么办？**

A：保持现状。如果空仓就继续空仓，如果持仓就继续持仓。

**Q4：可以调整买卖阈值吗？**

A：可以。阈值越高，信号越少但更可靠；阈值越低，信号越多但可能误判。
